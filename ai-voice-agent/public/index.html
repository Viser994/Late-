1

Configure Environment

Copy .env.example to .env and fill in your credentials.

cp .env.example .env
2

Get Your API Keys

You'll need accounts with:

  • Twilio — phone number + auth token
  • OpenAI — API key
3

Expose Your Server

Use ngrok to expose your local server to the internet for Twilio webhooks.

ngrok http 3000
4

Configure Twilio Webhook

In your Twilio phone number settings, set the "A call comes in" webhook to:

https://<your-ngrok-url>/call/incoming

Also set the Status Callback URL to:

https://<your-ngrok-url>/call/status
5

Set Up Call Forwarding

On your business phone, enable "No Answer Forwarding" to your Twilio number. The AI will pick up when you miss the call.

6

Start the Server

npm start

Your AI voice agent is now live and ready to handle missed calls!
